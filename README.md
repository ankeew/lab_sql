# SQL Practice

Учебные SQL-запросы, демонстрирующие ключевые навыки владения postgresql:
- Агрегация и группировка (`GROUP BY`, `COUNT`, `AVG`, `MIN`, `MAX`)
- Работа с текстом и регулярными выражениями (`REGEXP_REPLACE`, `~`)
- Извлечение информации из неструктурированных полей
- Финансовый анализ (топ операций, средние значения)
- Оконные функции и сложные условия фильтрации

## Структура проекта

- `schema.sql` — схема всех таблиц
- `sample_data.sql` — примеры данных (без персональной информации)
- `lab_01_salary_analysis.sql` — анализ зарплат по должностям
- `lab_02_email_processing.sql` — обработка email-адресов
- `lab_03_lastname_filtering.sql` — фильтрация по шаблону фамилий
- `lab_04_tech_extraction.sql` — извлечение технологий из описания
- `lab_05_financial_report.sql` — финансовый отчёт по банковским операциям

## Как запустить

1. Убедитесь, что у вас установлен **PostgreSQL**.
2. Создайте новую базу данных (например, `sql_practice`).
3. Подключитесь к ней через **любой SQL-клиент**:
   - **psql** (терминал)
   - **DBeaver**, **pgAdmin**, **DataGrip**, **VS Code + расширение**
4. Последовательно выполните содержимое файлов:
   - Сначала `schema.sql` (создание таблиц)
   - Затем `sample_data.sql` (вставка данных)
   - Потом любой из `lab_*.sql` (ваши запросы)
